# Quiniela Joan Santos — versión rápida solo Streamlit

Esta versión usa SQLite local y no requiere Supabase ni Secrets.

## Publicación
1. Sube todos los archivos a GitHub.
2. En Streamlit Community Cloud selecciona `app.py`.
3. No agregues credenciales de Supabase.
4. Despliega o reinicia la aplicación.

## Importante sobre los datos
Streamlit Community Cloud no garantiza almacenamiento permanente del archivo `quiniela.db`. Un reinicio, redeploy o cambio de servidor puede borrar la base local. Descarga respaldos periódicos o usa esta versión solo si aceptas ese riesgo.
